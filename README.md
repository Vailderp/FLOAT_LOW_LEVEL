# FLOAT_LOW_LEVEL

```cpp
int main()
{
  // Знак = 0
  // Мантисса = 01111111
  // Экспонента = 10000000000000000000000u
  __s32ll a = 0b0'01111111'10000000000000000000000u;

  // Вывод получившегося числа
  std::cout << get_info(a) << std::endl;
}
// Консоль:
// 1) 1.5f = 0 01111111 10000000000000000000000
// 2) Is a positive number
// 3) Exponent = 127
```
